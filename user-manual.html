command line tools

  • eputil is the new harvester, it talks to the REST API and can return JSON documents, XML or EPrints documents available from the REST API. It can be piped into a dataset if needed
  • epfmt is a command line utility to pretty print EPrints XML and convert to/from JSON
    • in the process of pretty printing it also validates the EPrints XML against the eprinttools Go package definitions
  • doi2eprintxml is a CaltechAUTHORS centric DOI to EPrint XML document generator
  • ep3apid is a web service that runs on localhost providing an extended EPrints 3 API via interaction with EPrints MySQL database(s)
  • ep3harvester harvests EPrints 3.x repository into a dataset collection
